Unity + Meta Quest 開発設定ガイド

インタラクティブなガイドで、設定手順を確実にこなしましょう。

1Unityプロジェクトの新規作成

  1. Unity Hubを起動し、「New project」ボタンをクリックします。
  2. テンプレートから「3D (Built-In Render Pipeline)」を選択します。
  3. プロジェクト名と保存場所を設定し、「プロジェクトを作成」ボタンをクリックします。
  1. Unityエディタが起動するまで待ちます。

2Meta XR Core SDKの準備

  1. Unityメニューから「ウィンドウ」>「アセットストア」を選択し、「Search online」をクリックします。
  2. ブラウザで開いたアセットストアで、以下のワードを検索します。
    Meta XR Core SDK
  1. 検索結果から該当アセットを選択し、「Add to My Assets」をクリックし、次に「Accept」をクリックします。

3ビルド設定とプレイヤー設定

  1. Unityメニューから「ファイル」>「ビルド設定 (Build Settings)」を開きます。
  2. プラットフォームから「Android」を選択し、「プラットフォームを切り替え (Switch Platform)」をクリックします。
  3. 「プレイヤー設定 (Player Settings)」を開き、以下の項目を設定します。
  • Company Name: `DefaultCompany`以外の名前に設定 （ヒント: 日本語は非推奨）
  • Product Name: 任意のプロジェクト名
  • Other Settings > Color Space: `Linear` （より正確な色表現）
  • Other Settings > Auto Graphics API: チェックを外す
  • Other Settings > Graphics API: リストから `Vulkan` を選択して削除 （安定性向上）
  • Other Settings > Minimum API Level: `Android 10.0 'Q' (API level 29)` 以上を推奨
  • Other Settings > Scripting Backend: `Mono` （開発中のビルド時間短縮）
  • Other Settings > Api Compatibility Level: `.NET Framework` （C#の機能拡張）

4XRプラグインの有効化

  1. 「プレイヤー設定」画面の左メニューから「XR Plug-in Management」をクリックします。
  2. 「Install XR Plugin Management」ボタンを押します。
  3. インストール後、Androidタブが選択されていることを確認し、「Oculus」にチェックを入れます。
  1. 設定が完了したら、各設定ウィンドウを閉じます。

5SDKインポートとシーン設定

  1. Unityメニューから「ウィンドウ」>「パッケージマネージャー」を開きます。
  2. 上部のドロップダウンを「マイアセット (My Assets)」に変更します。
  3. リストから「Meta XR Core SDK」を選択し、「インポート」または「インストール」します。
  4. 再起動を促されたら「Restart Editor」を押します。
  5. `Assets`の検索窓で以下のプレハブを検索し、Hierarchyビューにドラッグ＆ドロップします。
    OVRCameraRig
  1. 元からあった「Main Camera」オブジェクトをHierarchyビューから削除します。

6インタラクションの設定 (推奨)

注意: 必ず Step 5 (Meta XR Core SDK) を完了させてからこの手順に進んでください。
  1. アセットストアで「Meta XR Interaction SDK」を検索し、同様にプロジェクトに追加・インポートします。
  2. Unityメニューから「Meta XR Tools」>「Project Setup Tool」を選択します。
  3. 開いたウィンドウの「Fix All」ボタンをクリックして、設定を自動修正します。
